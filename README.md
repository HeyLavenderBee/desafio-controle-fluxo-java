## Controle de Fluxo
Desafio de sistema básico de controle de fluxo de input de dados.
<br>
Programa consiste de o usuário digitar dois parâmetros de números inteiros, subtrai eles e faz uma contagem de um até o resultado da subtração. Caso o primeiro parâmetro seja maior que o primeiro, lança uma exceção personalizada: `ParametrosInvalidosException`, caso contrário, o programa roda normalmente.
<br>
Foi trabalhado o uso de exceções personalizadas, métodos e trycatch.
